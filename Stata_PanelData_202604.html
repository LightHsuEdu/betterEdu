Stata 面板数据分析

1. 安装外部命令

ssc install ivreg2, replace
ssc install xtoverid, replace
ssc install xttest1, replace
ssc install xttest2, replace
ssc install xttest3, replace
ssc install xtcsd, replace
ssc install xtserial, replace
ssc install xtscc, replace
ssc install ftools, replace
ssc install reghdfe, replace
ssc install outreg2, replace
ssc install estout, replace
ssc install winsor2, replace
ssc install coefplot, replace

2. 启动 Stata，打开数据文件

3. 创建面板结构

 3.1 检查变量类型（需要修改：个体变量为 id ，时间变量为 year ，数值型）

describe id year

 3.2 创建面板结构

xtset id year

4. 数据处理

 4.1 定义 被解释变量与解释变量（需要修改：YOUR_DEP_VAR ，YOUR_INDEP_VARS）

local YYO YOUR_DEP_VAR
local XXO YOUR_INDEP_VARS
local YY `YYO'
local XX `XXO'
local XX_comma : subinstr local XX " " ", ", all

 4.2 检查缺失值

misstable summarize `YY' `XX'

 可选： 删除缺失值

drop if missing(`YY', `XX_comma')

 可选： 处理异常值（两种方法选择其一）

    1. 缩尾处理：进行 1% 和 99% 双侧缩尾
winsor2 `YY' `XX', cuts(1 99) replace
    2. 截尾处理：删除小于1%分位数和大于99%分位数的样本
winsor2 `YY' `XX', cuts(1 99) trim replace
drop if missing(`YY', `XX_comma')

5. 描述性统计

 5.1 整体描述

sum `YY' `XX'

 5.2 组内/组间描述

xtsum `YY' `XX'

 5.3 相关系数矩阵

pwcorr `YY' `XX', sig

 可选： 时间趋势图（需要修改：YOUR_VAR）

preserve
    collapse (mean) mean_y=YOUR_VAR (semean) se_y=YOUR_VAR, by(year)
    gen upper = mean_y + 1.96*se_y
    gen lower = mean_y - 1.96*se_y
    twoway (rarea upper lower year, color(gray%30)) (line mean_y year, lcolor(blue) lwidth(medium)), title("YOUR_VAR 年度平均趋势") xtitle("年份") ytitle("平均 YOUR_VAR") legend(order(2 "平均值" 1 "95% CI")) scheme(s1mono) name(timeVarPlot, replace)
restore

 可选： 变量关系图（散点 + 拟合线）

foreach x of local XX {
    twoway (scatter `YY' `x', mcolor(blue%20) msize(vsmall)) (lfit `YY' `x', lcolor(red)), title("`YY' 与 `x' 的关系") xtitle("`x'") ytitle("`YY'") name(corr_`x', replace)
}

6. 判别与诊断检验

 6.1 Hausman检验： Prob>chi2 < 0.05 选择固定效应模型 (FE)

xtreg `YY' `XX', fe
est store FE_plain
xtreg `YY' `XX', re
est store RE_plain
hausman FE_plain RE_plain, sigmamore

 6.2 稳健或聚类标准误Hausman检验（Sargan-Hansen 检验）： P-value (Sargan-Hansen stat) < 0.05 选择固定效应模型 (FE)

xtreg `YY' `XX', re vce(cluster id)
xtoverid

 6.3 Breusch-Pagan LM 检验： Prob>chibar2 < 0.05 随机效应模型（RE）优于Pooled

xtreg `YY' `XX', re
xttest0

 6.4 F检验(固定效应)： Prob>F < 0.05 固定效应模型（FE）优于Pooled

xtreg `YY' `XX', fe

 6.5 异方差检验(修正的 Wald 检验，要求 N < T)： Prob>chi2 < 0.05 存在异方差

xtreg `YY' `XX', fe
xttest3

 6.6 截面相关检验(Pesaran CD 检验)： Pr<0.05 存在截面相关

xtreg `YY' `XX', fe
xtcsd, pesaran abs

 6.7 截面相关检验(Breusch-Pagan LM 检验，仅适用于 N < T)： Prob>chi2 < 0.05 存在截面相关

xtreg `YY' `XX', fe
xttest2

 6.8 序列相关检验(Wooldridge 检验)： Prob>F < 0.05 存在序列相关

xtserial `YY' `XX'

 6.9 序列相关检验(Baltagi-Li 联合检验)： Serial Correlation Pr>chi2(1) < 0.05 存在序列相关，Joint Test Pr>chi2(2) < 0.05 存在个体效应和/或序列相关

xtreg `YY' `XX', re
xttest1

 6.10 面板单位根检验(Fisher-ADF 检验)： Prob < 0.05 至少部分截面平稳；Prob > 0.05 存在单位根（非平稳）

foreach v in `YY' `XX' {
    quietly xtunitroot fisher `v', dfuller lags(1)
    local p_P  = r(p_P)
    local p_Pm = r(p_Pm)
    local p_Z  = r(p_Z)
    if `p_P' > 0.05 {
        display as error "警告: 变量 `v' 可能存在单位根" " | 逆卡方P统计量 p=" %5.3f `p_P' " | 修正逆卡方Pm统计量 p=" %5.3f `p_Pm' " | 逆正态Z统计量 p=" %5.3f `p_Z'
    }
    else {
        display as result "通过: 变量 `v' 平稳" " | 逆卡方P统计量 p=" %5.3f `p_P' " | 修正逆卡方Pm统计量 p=" %5.3f `p_Pm' " | 逆正态Z统计量 p=" %5.3f `p_Z'
    }
}
 🠊 提示：若多个变量不平稳，建议执行下方 [可选] 步骤或进行协整检验。

 6.11 面板协整检验(Kao 协整检验)： P < 0.05 存在协整关系

xtcointtest kao `YY' `XX'
 🠊 提示：仅当所有 YY 和 XX 均不平稳时，才需检验是否存在协整关系；若 P < 0.05，即便变量不平稳，也可直接使用水平值（或对数）建模，无需差分。

 可选： 面板数据可能存在非平稳性，直接回归可能导致伪回归（四种方法选择其一）

    1. 变量平稳，或不平稳但有协整关系（无需操作）
    2. 变量不平稳（对数化处理，尺度变换并减弱异方差，反映弹性/长期均衡；对数化后仍然需要复检平稳性/协整）
local YY `YYO'
local XX `XXO'
foreach v in `YY' `XX' {
    capture confirm variable ln_`v'
    if _rc {
        quietly gen double ln_`v' = .
    }
    quietly replace ln_`v' = ln(`v') if `v' > 0
    quietly replace ln_`v' = . if `v' <= 0 | missing(`v')
    quietly count if `v' <= 0 & !missing(`v')
    if r(N) > 0 {
        noisily di as error "变量 `v' 含有 `r(N)' 个非正值：ln_`v' 已自动设为缺失。"
    }
}
local YY ln_`YY'
local new_XX ""
foreach v in `XX' {
    local new_XX `new_XX' ln_`v'
}
local XX `new_XX'
local XX_comma : subinstr local XX " " ", ", all
drop if missing(`YY', `XX_comma')
display "对数化处理后，当前的被解释变量为: `YY'"
display "对数化处理后，当前的解释变量列表为: `XX'"
    3. 变量不平稳且无协整关系（必须使用一阶差分以消除伪回归风险，反映短期波动）
local YY `YYO'
local XX `XXO'
foreach v in `YY' `XX' {
    capture confirm variable d_`v'
    if _rc {
        quietly gen double d_`v' = .
    }    
    quietly replace d_`v' = D.`v'
}
local YY d_`YY'
local new_XX ""
foreach v in `XX' {
    local new_XX `new_XX' d_`v'
}
local XX `new_XX'
local XX_comma : subinstr local XX " " ", ", all
drop if missing(`YY', `XX_comma')
display "差分处理后，当前的被解释变量为: `YY'"
display "差分处理后，当前的解释变量列表为: `XX'"
    4. 变量不平稳且无协整关系（进阶：对数一阶差分，反映增长率并确保平稳性）
local YY `YYO'
local XX `XXO'
foreach v in `YY' `XX' {
    capture confirm variable ln_`v'
    if _rc {
        quietly gen double ln_`v' = .
    }
    quietly replace ln_`v' = ln(`v') if `v' > 0
    quietly replace ln_`v' = . if `v' <= 0 | missing(`v')
    quietly count if `v' <= 0 & !missing(`v')
    if r(N) > 0 {
        noisily di as error "变量 `v' 含有 `r(N)' 个非正值：ln_`v' 已自动设为缺失。"
    }
    capture confirm variable d_ln_`v'
    if _rc {
        quietly gen double d_ln_`v' = .
    }
    quietly replace d_ln_`v' = D.ln_`v'
}
local YY d_ln_`YY'
local new_XX ""
foreach v in `XX' {
    local new_XX `new_XX' d_ln_`v'
}
local XX `new_XX'
local XX_comma : subinstr local XX " " ", ", all
drop if missing(`YY', `XX_comma')
display "对数差分处理后，当前的被解释变量为: `YY'"
display "对数差分处理后，当前的解释变量列表为: `XX'"

 可选： 残差 vs 拟合值图

quietly xtreg `YY' `XX', fe
capture drop e yhat
predict e, e
predict yhat, xb
twoway (scatter e yhat, mcolor(blue%30) msize(vsmall)), yline(0, lcolor(red)) title("残差 vs 拟合值 (异方差检查)") ytitle("残差 (Residuals)") xtitle("拟合值 (Fitted Values)") note("如果散点呈喇叭状发散，说明存在异方差") name(testHetero, replace)

 可选： 残差滞后散点图

quietly xtreg `YY' `XX', fe
capture drop e e_lag
predict e, e
gen e_lag = L.e
twoway (scatter e e_lag, mcolor(blue%30) msize(vsmall)) (lfit e e_lag, lcolor(red)), title("残差滞后散点图(序列相关检查)") ytitle("当前残差 (e_t)") xtitle("滞后残差 (e_t-1)") note("如果红线明显倾斜，说明存在序列相关") name(testSerial, replace)

 可选： 年度平均残差图

capture drop e

    collapse (mean) mean_e=e, by(year)
    twoway (line mean_e year, lcolor(blue)), yline(0) title("年度平均残差图(截面相关检查)") ytitle("平均残差") xtitle("年份") note("如果曲线呈现明显波浪状，说明存在截面相关") name(testCrosSec, replace)

7. 基本模型

 7.1 Pooled 混合回归（单向聚类）

reg `YY' `XX', vce(cluster id)
est store myOLS_CT0CL1

 7.2 Pooled 混合回归（时间固定效应+单向聚类）

reg `YY' `XX' i.year, vce(cluster id)
est store myOLS_CTtimeCL1

 7.3 Fixed Effects Model 固定效应模型（个体固定效应+单向聚类）

xtreg `YY' `XX', fe vce(cluster id)
est store myFE_CTidCL1

 7.4 Fixed Effects Model 固定效应模型（双向固定效应+单向聚类）

xtreg `YY' `XX' i.year, fe vce(cluster id)
est store myFE_CT2CL1

 7.5 Random Effects Model 随机效应模型（个体随机效应+单向聚类）

xtreg `YY' `XX', re vce(cluster id)
est store myRE_CRidCL1

 7.6 Random Effects Model 随机效应模型（个体随机效应+时间固定效应+单向聚类）

xtreg `YY' `XX' i.year, re vce(cluster id)
est store myRE_CR2CL1

8. 补充模型

 8.1 Driscoll–Kraay稳健标准误（SCC）

xtscc `YY' `XX', fe lag(2)
est store mySCC

 8.2 Driscoll–Kraay稳健标准误（SCC）+双向固定效应

xtscc `YY' `XX' i.year, fe lag(2)
est store mySCC_CT2

 8.3 面板校正标准误（PCSE）

xtpcse `YY' `XX', corr(psar1) pairwise
est store myPCSE

 8.4 面板校正标准误（PCSE）+双向固定效应

xtpcse `YY' `XX' i.id i.year, corr(psar1) pairwise
est store myPCSE_CT2

 8.5 双向固定效应模型（个体+时间双向聚类）

reghdfe `YY' `XX', absorb(id year) vce(cluster id year)
est store myReghdfe
